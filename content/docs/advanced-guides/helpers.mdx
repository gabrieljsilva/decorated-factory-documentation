---
title: Helpers
description: AutoIncrement and UUID helpers for convenient identifiers
---

Use helpers with `@FactoryType` to generate common identifiers.

| Helper | Produces | Example |
| --- | --- | --- |
| `AutoIncrement` | Sequential integers starting at 1 (per builder) | `1, 2, 3…` |
| `UUID` | RFC‑4122 v4 UUID strings | `d7f3e429‑9d5b‑42f9‑b7de‑8ba0e50bc9f6` |

```ts
class Task {
  @FactoryType(() => AutoIncrement)
  id: number;

  @FactoryType(() => UUID)
  taskId: string;
}

const task = factory.one(Task).make();
```

<Callout>
  `AutoIncrement` is not supported for arrays. Generating an array of auto-incremented values will throw by design.
</Callout>
